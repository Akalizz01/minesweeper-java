# 🧩 Minesweeper (Java)

Console-based Minesweeper game developed in Java as part of a university project.

## 📖 Description

This project is a fully functional implementation of the classic **Minesweeper** game, played in the console. It demonstrates core programming concepts such as game logic, user interaction, and data handling using Java.

The game allows players to uncover cells, place flags, and avoid hidden mines while tracking their performance.

---

## 🚀 Features

* 🎮 Complete Minesweeper gameplay
* 🚩 Flag system to mark suspected mines
* 💡 Hint system to assist the player
* 🕒 Game timer / tracking
* 📊 History of recent games (results and time)
* ⚠️ Cheat/debug mode
* 🧠 Input validation and game state management

---

## 🛠️ Technologies Used

* **Java**
* Console / Terminal interface
* Object-Oriented Programming (OOP)

---

## ▶️ How to Run

1. Compile the project:

   ```bash
   javac *.java
   ```

2. Run the game:

   ```bash
   java Main
   ```
